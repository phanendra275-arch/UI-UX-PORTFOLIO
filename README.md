## 🧠 Design Process

This portfolio demonstrates a structured UX workflow:

1️⃣ Empathize  
Understanding user needs through research and interviews.

2️⃣ Define  
Creating user personas, pain points, and problem statements.

3️⃣ Ideate  
Generating solutions using brainstorming and user flows.

4️⃣ Design  
Wireframes → High Fidelity UI → Interactive prototypes.

5️⃣ Test  
Usability testing and iteration based on feedback.
