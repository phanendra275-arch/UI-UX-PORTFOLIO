WHAT IS CONNECTSPHERE?

ConnectSphere is a social media platform built around meaningful communities — not individual vanity metrics. Users join "Spheres" (topic-based communities) and engage through posts, live Q&As, and collaborative projects.

Core insight: people don't want more followers — they want to find their people.

🎯
Primary Goal
Reduce isolation, increase genuine connection
👥
Target Users
18–35, interest-driven
📱
Platforms
iOS · Android · Web
🏆
Key Outcome
87% satisfaction in testing
— 02  Problem & Solution

THE PROBLEM I SOLVED

Existing social media optimises for engagement time, not user wellbeing — creating a toxic cycle of comparison, doomscrolling, and shallow interaction.

😔
The Problems
  • Algorithm-driven feeds promote outrage over connection
  • Follower counts used as proxy for self-worth
  • No way to find niche interest communities
  • Content from strangers drowns close connections
  • No digital wellbeing tools built in
✨
ConnectSphere's Solutions
  • Interest-based Spheres replace algorithmic feeds
  • Community reputation replaces follower counts
  • Sphere matching surfaces relevant communities
  • Separate close-friends and community feeds
  • Built-in mindful usage dashboard
— 03  User Research

— 01  Overview

WHAT IS FLOWBUDGET?

FlowBudget is a personal finance app designed around speed and clarity. The core promise: logging any transaction in under 7 seconds. Traffic-light budget bars give instant status without reading numbers.

⚡
Primary Goal
Sub-7s transaction logging
👥
Target Users
22–35 yr, salaried, India
📱
Platforms
iOS · Android · Web
🏆
Key Outcome
6.4s avg, 0 mislogged
— 02  Problem & Solution

THE PROBLEM I SOLVED

Existing finance apps require too many taps, too many decisions, and deliver no instant budget clarity — so users log inconsistently and lose financial awareness.

😤
The Problems
  • Average transaction log took 38+ seconds
  • Complex category hierarchies caused confusion
  • No immediate visual of budget status
  • Users abandoned mid-session, causing data gaps
  • No difference between phone/web experience
⚡
FlowBudget's Solutions
  • Numpad-first UI — amount in 2 taps
  • 12 quick-pick categories with emoji icons
  • Traffic-light bars: green/amber/red at a glance
  • One-swipe confirmation, no modal dialogs
  • Sidebar web dashboard synced in real time
— 03  User Research
